Intégrer un forum bbPress 2.0 sur votre blog WordPress

7 septembre 2011 - Aurélien Denis - Outils pratiques, Tutoriels WordPress - bbPress, Custom Post Types, Forum

bbPress 2.0 est disponible gratuitement au téléchargement… depuis le répertoire de plugins de WordPress. Il est désormais possible de créer des forums depuis le back office de WordPress en installant bbPress comme n’importe quel plugin.

Cette nouvelle devrait en ravir plus d’un tant la création de forums avec WordPress est une question récurrente à laquelle nous avions déjà répondu par 3 tutoriels :

  1. Tutoriel #1 : Installation de Simple Press Forum ;
  2. Tutoriel #2 : Configuration de l’administration du forum ;
  3. Tutoriel #3 : Intégrer un forum bbPress dans votre blog WordPress 3.0 ;

Aujourd’hui, la donne change complètement comme nous allons le voir en détails dans ce tutoriel.

Nouveautés de bbPress 2.0

bbPress est donc de devenu un plugin parfaitement intégré au back office de WordPress. Léger et performant, des extensions verront bientôt le jour pour en étendre ses fonctionnalités.

Utilisation de shortcodes

Les shortcodes sont des bouts de code entre crochets que l’on peut saisir directement dans l’éditeur de contenu de WordPress. bbPress dispose d’une liste conséquente de shortcodes vous permettant d’afficher très facilement vos différents forums, sujets ou autre.

Je vous laisse imaginer le potentiel de tels shortcodes : plus besoin de se compliquer la vie à créer des templates en PHP.

Compatibilité avec les thèmes

Dans le même ordre idée que les shortcodes, bbPress s’intègre parfaitement avec les thèmes par défaut de WordPress – TwentyTen et TwentyEleven. Pour information, bbPress utilise le principe des child themes, une méthode propre à WordPress qui prouve là encore l’intégration très poussée entre ces 2 outils.

Pour autant, si vous n’utilisez pas les thèmes par défaut comme base de travail à un thème personnalisé : pas de soucis ! Une solution est disponible sur le site officiel : bbPress 2.0 – Theme compatibility.

Une fois de plus, il y a fort à parier qu’un plugin verra le jour pour vous faciliter le travail.

Widgets bbPress pour WordPress

Des widgets sont également disponibles pour afficher les derniers sujets ou encore les différentes rubriques : une fonction demandée de longue date par la communauté.

Processus d’intégration simplifié de bbPress 2.0

Installation de bbPress 2.0

Rendez-vous pour commencer dans le menu Extensions puis cliquez sur Ajouter.

Faites une recherche en indiquant le mot-clé « bbpress » puis faites Installer maintenant.

3 nouveaux types de contenus apparaissent alors – on parle de custom post types :

  • Forums : répertorie l’ensemble des forums créés. Vous pouvez définir des catégories pour une meilleure classification ;
  • Sujets : répertorie l’ensemble des sujets créés ou topics ;
  • Réponses : répertorie l’ensemble des réponses apportées par les internautes ;

Cette nouvelle gestion présente notamment l’avantage de pouvoir gérer l’intégralité du contenu directement depuis le back office de WordPress. Les comptes utilisateurs sont donc pris en charge et il n’existe plus cette séparation entre d’un côté WordPress et de l’autre bbPress.

Configuration de bbPress

Les options sont directement présentes sous le menu Réglages puis Forums.

La plupart des options sont suffisamment explicites pour ne pas avoir à les détailler ici. On s’attardera toutefois sur la gestion des permaliens ou slugs que l’on peut modifier à sa guise pour :

  • Identifiant du forum
  • Identifiant du sujet
  • Identifiant du tag du sujet
  • Identifiant de réponse
  • Base d’utilisateurs
  • Vue de la base

Télécharger la traduction française

Vous pouvez obtenir la traduction française de bbPress 2.0, qui est proposée par le site Aytechnet :

Quelques erreurs sont malgré tout présentes mais remercions l’auteur pour son travail. Et signalons que tout un chacun peut opérer sa propre traduction en éditant le fichier bbpress.pot présent dans le dossier /wp-content/plugins/bbpress/bbp-languages de votre serveur FTP.

Ce type de fichier nécessite l’usage du logiciel Poedit, multiplateforme et simple d’utilisation. Il vous suffira d’enregistrer les fichiers bbpress.mo dans le même répertoire pour appliquer les changements.

Pour information, Poedit est également utilisé pour traduire les thèmes de WordPress.

De par ce constat très éloquent, bbPress 2.0 a donc tout naturellement été choisi pour propulser les forums de WordPress Channel ! Je recommande toujours bbPress mais nous ne n’utilisons plus de forums sur ce site.
